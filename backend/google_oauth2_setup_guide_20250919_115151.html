1

Google Cloud Console 접속

Google Cloud Console에 접속하여 새 프로젝트를 생성합니다.

https://console.cloud.google.com/

프로젝트 정보

  • 프로젝트 이름: newsbot-kr-analytics
  • 프로젝트 ID: newsbot-kr-analytics-20250919
  • 설명: 3차원 통합 선거 예측 시스템 - 데이터 분석 및 시각화
2

필수 API 활성화

다음 API들을 활성화해야 합니다:

Google Drive API

파일 업로드, 폴더 관리, 권한 설정

drive.googleapis.com

Google Sheets API

CSV → Sheets 변환, 데이터 조작

sheets.googleapis.com

Google Data Studio API

대시보드 자동 생성 (베타)

datastudio.googleapis.com

Google Maps API

지리적 시각화, 좌표 변환

maps-backend.googleapis.com

API 활성화 방법

  1. 좌측 메뉴에서 "API 및 서비스" → "라이브러리" 클릭
  2. 각 API를 검색하여 "사용 설정" 클릭
  3. 모든 API 활성화 완료 후 다음 단계 진행
3

OAuth2 클라이언트 ID 생성

OAuth2 인증을 위한 클라이언트 ID를 생성합니다.

https://console.cloud.google.com/apis/credentials

설정 정보

  • 애플리케이션 유형: 데스크톱 애플리케이션
  • 이름: NewsBot-KR OAuth Client
  • 승인된 리디렉션 URI:
    • http://localhost:8080/oauth2callback
    • urn:ietf:wg:oauth:2.0:oob
4

credentials.json 다운로드

생성된 OAuth2 클라이언트의 JSON 파일을 다운로드합니다.

다운로드 위치

/Users/hopidaay/newsbot-kr/backend/credentials.json

다운로드한 JSON 파일을 위 경로에 credentials.json 이름으로 저장하세요.

5

OAuth2 동의 화면 설정

OAuth2 동의 화면을 설정하고 테스트 사용자를 추가합니다.

https://console.cloud.google.com/apis/credentials/consent

중요 설정

  • 사용자 유형: 외부 (External)
  • 앱 이름: NewsBot-KR Analytics
  • 테스트 사용자 추가: justbuild.pd@gmail.com
  • 범위(Scopes): 기본 프로필, 이메일, Drive, Sheets
6

OAuth2 인증 테스트

설정이 완료되면 OAuth2 인증을 테스트합니다.

테스트 명령어

cd /Users/hopidaay/newsbot-kr/backend && python3 google_oauth2_setup.py --test

위 명령어를 실행하면 브라우저가 열리고 OAuth2 인증 플로우가 시작됩니다.
