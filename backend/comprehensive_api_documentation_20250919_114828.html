🔐 OAuth2 설정 가이드

1

Google Cloud Console 프로젝트 생성

console.cloud.google.com에서 새 프로젝트 생성

🔗 https://console.cloud.google.com/projectcreate
2

API 활성화

Drive API, Sheets API, Data Studio API 활성화

3

OAuth2 클라이언트 ID 생성

사용자 인증 정보 → OAuth 2.0 클라이언트 ID → 데스크톱 애플리케이션

🔗 https://console.cloud.google.com/apis/credentials
4

credentials.json 다운로드

생성된 클라이언트 ID의 JSON 파일 다운로드

5

OAuth2 동의 화면 설정

테스트 사용자에 justbuild.pd@gmail.com 추가

🔗 https://console.cloud.google.com/apis/credentials/consent
